## What this sample does ?

**BlackBoxMacro** is a simple macro that creates a function that serializes an object of type T to json.   

**WhiteBoxMacro** creates an object which depending on its only parameter, could have either fields of { `a` ,`b` and `c` } or { `z` }

## How to run ?

Simply run `BlackBoxMacroApp` or `WhiteBoxMacroApp`.
